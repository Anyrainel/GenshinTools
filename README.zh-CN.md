## 🛠️ 包含工具

### 圣遗物过滤助手

**定制你自己的圣遗物过滤器！**

每个人玩原神的方式都不一样。这个工具可以让你：

- **分角色配装**：根据你的具体需求为每个角色设定配装方案（套装 + 词条）。
- **智能合并**：将所有的配装需求合并成优化后的过滤器配置。
- **分享与导入**：轻松分享你的配置或导入社区的方案。

### 账号数据分析 (Account Analytics)

**可视化你的账号进度。**

- **数据导入**：通过 **GOOD** 文件（来自扫描器）或 **Enka.Network**（UID）导入数据。
- **圣遗物评分**：基于自定义权重自动为你的角色圣遗物评分。
- **库存概览**：查看账号内圣遗物和武器的分布情况。

### 角色强度榜制作器

**创建和管理角色强度榜。**

一个现代化的交互式强度榜制作工具：

- **按元素分类**：在每一层级内，角色按元素自动排序，更加清晰。
- **保存/加载**：将你的强度榜保存为 JSON 文件，随时加载修改。
- **拖拽操作**：简单易用的界面。
- **图片导出**：一键生成高清图片分享你的榜单。

### 武器榜单制作器

**按类型排列武器。**

在同类武器（单手剑、双手剑等）中进行对比，支持按**副词条**（如暴击率、元素精通）和星级进行高级筛选。

- **图片导出**：分享你的武器排行。

### 队伍构建器

**可视化构筑你的理想队伍。**

- **直观规划**：在卡片式布局中选择角色、武器和圣遗物。
- **智能交互**：带有筛选功能的智能选择器（例如仅显示角色可用的武器）。
- **管理功能**：创建、复制并管理多个队伍配置。

## 🖥️ 桌面版应用

**原生性能与离线支持。**

本应用使用 **Tauri** 构建，支持作为 Windows 桌面应用运行，无需浏览器即可使用，且性能更加出色。

## 💻 技术栈

- **框架**: React + TypeScript
- **构建工具**: Vite
- **UI**: Tailwind CSS, shadcn/ui
- **状态管理**: Zustand
- **桌面端**: Tauri
