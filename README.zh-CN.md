## 🛠️ 包含工具

### 圣遗物过滤助手
**定制你自己的圣遗物过滤器！**

每个人玩原神的方式都不一样。这个工具可以让你：
- **分角色配装**：根据你的具体需求为每个角色设定配装方案（套装 + 词条）。
- **智能合并**：将所有的配装需求合并成优化后的过滤器配置。
- **分享与导入**：轻松分享你的配置或导入社区的方案。

### 角色强度榜制作器
**创建和管理角色强度榜。**

一个现代化的交互式强度榜制作工具：
- **按元素分类**：在每一层级内，角色按元素自动排序，更加清晰。
- **保存/加载**：将你的强度榜保存为 JSON 文件，随时加载修改。
- **拖拽操作**：简单易用的界面。

## 💻 技术栈

- **框架**: React + TypeScript
- **构建工具**: Vite
- **UI**: Tailwind CSS, shadcn/ui
- **状态管理**: Zustand / React Context
