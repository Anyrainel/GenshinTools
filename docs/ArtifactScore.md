# 圣遗物评分计算公式

## 属性公式

| 属性         | 计算公式                          |
| :----------- | :-------------------------------- |
| 暴击率       | 评分=暴击率*2*收益权重            |
| 暴击伤害     | 评分=暴击伤害*2*收益权重          |
| 元素精通     | 评分=元素精通*0.33*收益权重       |
| 元素充能效率 | 评分=元素充能效率*1.1979*收益权重 |
| 大攻击       | 评分=大攻击*1.33*收益权重         |
| 大生命       | 评分=大生命*1.33*收益权重         |
| 大防御       | 评分=大防御*1.06*收益权重         |
| 小攻击       | 评分=小攻击*0.398*0.5\*收益权重   |
| 小生命       | 评分=小生命*0.026*0.66\*收益权重  |
| 小防御       | 评分=小防御*0.335*0.66\*收益权重  |

其中大攻击、大生命、大防御为百分比属性，小攻击、小生命、小防御为固定值属性。收益权重因不同角色而异。
